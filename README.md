# ALA K. P. & Associates Advocates Website

## Project Overview

**ALA K. P. & Associates Advocates** is a dynamic and client-focused law firm based in Nairobi, Kenya, providing innovative, practical and results-oriented legal solutions to individuals, businesses financial institutions, non-governmental organizations, public sector entities and educational institutions.
The Firm is founded on the principles of professionalism, integrity, responsiveness and excellence. 
We combine strong legal expertise with a deep understanding of our clients' business and operational environments, enabling us to provide commercially sound and legally effective solutions. Our practice spans a wide range of legal disciplines including litigation, dispute resolution, banking and finance, debt recovery, employment law, conveyancing and real estate, commercial transactions, regulatory compliance, corporate governance, company secretarial services, taxation, public policy and legal advisory services. With extensive experience representing both individual and corporate clients, the Firm has developed a reputation for delivering practical legal solutions while maintaining the highest standards of professional ethics and client service.

The website is able to show all the services offered by the firm and making to easier for the client to be able to access the necessary tools they need. This will intergrate The firm"s services breaching the gap between the client and associates hence quick services are able to be provided.

## Installation

1. Clone the repository

```bash
git clone https://github.com/Sawanda-Seth/ala-kp-associates-website.git
```

2. Open the project folder

```bash
cd Lawfirm-website
```

3. Open `index.html` in your browser.

---

## Running the Project

Simply open the `index.html` file in your browser.

Alternatively, use the VS Code Live Server extension:

```bash
Right Click → Open with Live Server
```
